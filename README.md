# Readme do projeto

### Em caso de erro execute as seguintes linhas:

#### Para WINDOWS
* cd android 
* ./gradlew clean
* ./gradlew :app:bundleRelease
* npm install
* cd functions; npm install

#### Para linux UBUNTU
* chmod 755 android/gradlew
* npm install
* cd functions; npm install

#### No diretorio raiz
* react-native run-android
